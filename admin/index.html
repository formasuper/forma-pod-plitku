Ручное редактирование каталога

Измените позицию, скачайте обновленный файл и загрузите его в репозиторий вместо data/catalog.json.

Эта версия работает без сервера и паролей: изменения сохраняются в скачиваемый JSON-файл. Прямая запись в GitHub из браузера потребует отдельной OAuth-настройки.
